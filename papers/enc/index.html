enClosure: Group Communication via Encounter Closures

The enClosure project is part of a larger project addressing EbN: Encounter-based Networking. enClosure is a service providing group communication based on encounter closures for mobile and IoT applications. We implemented a prototype implementation for Android and the Microsoft Embedded Social Cloud platform. Using real-world traces, we showed that enClosure provides a privacy-preserving, secure platform for a wide range of group communication applications rangingfrom connecting attendees of a large event and virtual guest books to disseminating health risk warnings, lost-and-found, and tracing missing persons.

This project was done on a Fulbright grant to Germany.
